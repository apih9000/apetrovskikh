# Alexey Petrovskikh - Portfolio Site

## 🚀 Quick Start

This is a static site built with HTML, CSS, and vanilla JavaScript. No build process required.

## 🎨 Customization

### Update Content

- **Personal info**: Edit [index.html](index.html) sections
- **Colors**: Modify CSS variables in [styles.css](styles.css) (`:root` section)
- **Projects**: Add/edit project cards in the Projects section
- **Experience**: Update timeline in Experience section

### Sections

The site includes:
- 🎯 Hero section with current role and stats
- 📝 About section
- 💼 Value proposition (how you help companies)
- 🏢 Experience timeline
- 🚀 Notable projects
- 📧 Contact section

## 📱 Responsive Design

The site is fully responsive and optimized for:
- Desktop (1200px+)
- Tablet (768px - 1199px)
- Mobile (< 768px)

### Color Scheme

Current colors are in `styles.css`:
```css
:root {
    --primary-color: #2563eb;      /* Main blue */
    --secondary-color: #1e40af;     /* Darker blue */
    --text-dark: #1f2937;           /* Dark gray */
    --text-light: #6b7280;          /* Light gray */
    --bg-light: #f9fafb;            /* Background */
}
```
